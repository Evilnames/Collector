# Add a New Coffee Biome

Coffee is biome-driven in [coffee.py](../../../coffee.py). There are no named coffee "types" — each biome produces a unique flavor profile. Adding a new biome means adding it to `BIOME_FLAVOR_PROFILES`, `_CODEX_BIOMES`, and `BIOME_DISPLAY_NAMES`.

Individual beans are `CoffeeBean` dataclass instances generated by `CoffeeGenerator`. The generator reads `BIOME_FLAVOR_PROFILES` directly — no generator changes needed.

## Step 1 — Add a profile to BIOME_FLAVOR_PROFILES

Add the new biome key to `BIOME_FLAVOR_PROFILES` in [coffee.py](../../../coffee.py). Each attribute is a base value 0.0–1.0; the generator applies a small gaussian jitter per bean.

```python
BIOME_FLAVOR_PROFILES = {
    ...
    "coral_reef": {
        "acidity":    0.90,   # bright and citrusy
        "body":       0.30,   # light
        "sweetness":  0.85,   # fruity sweet
        "earthiness": 0.05,   # almost none
        "brightness": 0.95,   # very floral
        "variety":    "arabica",   # "arabica" | "robusta" | "liberica"
    },
}
```

**variety** sets the bean's cultivar label:
- `"arabica"` — high acidity and brightness, lower earthiness
- `"robusta"` — high body and earthiness, lower acidity
- `"liberica"` — mid-range across all attributes, unique flavour

## Step 2 — Add to _CODEX_BIOMES

`COFFEE_TYPE_ORDER` is built from `_CODEX_BIOMES`. Add the new key there:

```python
_CODEX_BIOMES = ["tropical", ..., "rolling_hills", "coral_reef"]  # add here
```

The codex will then list all 5 roast levels for the new biome automatically.

## Step 3 — Add to BIOME_DISPLAY_NAMES

```python
BIOME_DISPLAY_NAMES = {
    ...
    "coral_reef": "Coral Reef",
}
```

## Flavor mechanics (no changes needed)

Flavor notes appear automatically based on which profile attributes exceed 0.6:

| Attribute | Notes drawn from (if > 0.6) |
|-----------|-----------------------------|
| `acidity` | citrus, lemon zest, bright cherry, green apple, tamarind, passion fruit |
| `body` | dark chocolate, molasses, walnut, heavy cream, black tea, toffee |
| `sweetness` | caramel, honey, ripe berry, brown sugar, vanilla, dried fig |
| `earthiness` | cedar, tobacco, forest floor, dried herb, mushroom, dark soil |
| `brightness` | floral, jasmine, bergamot, stone fruit, rose hip, elderflower |

## Verification checklist

- [ ] Biome key added to `BIOME_FLAVOR_PROFILES`
- [ ] Same key added to `_CODEX_BIOMES`
- [ ] Same key added to `BIOME_DISPLAY_NAMES`
- [ ] Biome key matches the key used in [biomes.py](../../../biomes.py)
- [ ] Harvest coffee cherries in the new biome — bean should generate and appear in the Coffee Codex
